## Como publicar (cPanel)

1. Fazer zip da pasta `site/`
2. Entrar no cPanel da hospedagem do domínio `refugio4elementos.com.br`
3. Acessar Gerenciador de Arquivos > `public_html`
4. Fazer upload do zip e extrair
5. Garantir que `index.html` está diretamente dentro de `public_html/`
